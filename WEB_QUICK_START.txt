FASTEST WAY TO RUN IN YOUR BROWSER:
================================================================================

STEP 1: Install Streamlit (one-time only)

   pip install streamlit requests

   OR

   pip install -r requirements_web.txt


STEP 2: Launch the app

   streamlit run app_streamlit.py


STEP 3: Browser opens automatically!

   The app will open at: http://localhost:8501

   If it doesn't open, manually go to that URL in your browser.


STEP 4: Start using it!

   - Select sport and division in the sidebar
   - Click "Fetch Events"
   - Click "Add" on events you want
   - Click "Download XML" to save

TROUBLESHOOTING:
================================================================================

Problem: "streamlit: command not found"
Solution: pip install streamlit

Problem: "Port 8501 already in use"
Solution: streamlit run app_streamlit.py --server.port 8502

Problem: App won't open in browser
Solution: Manually go to http://localhost:8501

Problem: No events showing
Solution: Check internet connection, try different date
